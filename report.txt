Project Report: OriginDrive Robot Simulation

This document explains what each file in this project does in simple terms.

--- 1. The ROS2 Package Setup Files ---
These files tell the computer how to build and organize the project so that it works as a standard ROS2 package.
* package.xml - Lists the name of the project ("origindrive") and the ROS2 tool libraries it needs to run (like rclpy and geometry_msgs).
* setup.py - The main instruction manual for installing the package. It tells ROS2 exactly where to look for the code and where the main starting point of the application is located.
* setup.cfg - A small configuration file that helps install the python scripts into the right, safe directory.

--- 2. The Robot Simulation Code (Inside the "origindrive" folder) ---
These files contain the actual math and logic that make the robot drive.

* main.py
This is the main "brain" or starting point of the application. It runs as a ROS2 "Node" (a program that talks to other ROS2 programs). When this runs, it calls all the other files below step-by-step to calculate the robot's route and drive it.

* path_smoother.py
The robot is given a set of jagged, straight-line target points (waypoints). This file uses math to smooth those sharp corners out, drawing a nice, drivable curved path for the robot to follow.

* trajectory_generator.py
Once we have a smooth path to drive on, this file figures out exactly how fast the robot is allowed to go. It plans out the speed (velocity) so the robot accelerates to its top speed safely and slows down when required.

* controller.py
This file handles the steering. It uses an algorithm called "Pure Pursuit." Imagine a dog chasing a rabbit—this code constantly looks at where the robot is right now, looks a little bit ahead at the target path, and figures out how much to turn the steering wheel to stay on track.

* visualiser.py
This file simply takes all the numbers generated by the step-by-step driving simulation and draws nice pictures (graphs and plots) so that we can clearly see the route the robot took and how fast it drove.
